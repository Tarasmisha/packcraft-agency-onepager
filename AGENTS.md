# Build Rules

Use the files in this repository as the only source of truth.

Primary sources:
- final-source.docx
- visual-rules.md

Task:
Build a faithful static PackCraft agency one-pager.

Audience:
Agency owners and agency operators only.

Do not:
- rewrite the offer from scratch
- compress or dry the copy
- broaden the audience
- mix in solo or in-house messaging
- invent testimonials, metrics, contacts, or links

Required sections:
- Hero
- Margin Math
- What You Actually Get
- Production Pack in Action
- Why This Is Not Just Claude + Prompts
- White-Label and Control
- How the Free Pilot Works
- Founder Proof
- Pricing and CTA
- Website/contact/footer strip

Required pricing logic:
- Free pilot on first client
- $800/client/month
- Setup for clients 2–3: $1,500/client
- Setup for clients 4+: $1,200/client
- Annual plans available on request

Required CTA:
Start a free pilot on one client

Technical output:
- index.html
- styles.css
- print-friendly A4 layout
- no external frameworks unless absolutely necessary
- high-fidelity copy
- restore any missing website/contact/footer layer
- if contact data is missing, show [TO CONFIRM EMAIL] and [TO CONFIRM DEMO LINK]
